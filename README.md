# 🔬 Cell Architecture Studio

Welcome to the Cell Architecture Studio! This is an interactive, web-based 3D biological viewer designed to let users explore microscopic cellular structures in realtime. Cycle through different specimen models, view detailed structural breakdowns, explore anatomical notes, and cross-compare biological profiles effortlessly.

## 📁 Repository Directory Checklist

To build or run this project seamlessly, ensure all the following files are sitting in your main folder directory without nested subfolders:

```text
├── index.html                  # Core application structure
├── style.css                   # Custom layout stylesheets
├── app.js                      # Central interaction engine
├── microscope.png              # Shared microscopy viewport graphic
│
├── animal cell.glb             # 3D Animal Cell Model
├── muscle cell.glb             # 3D Muscle Fiber Model
├── plant cell.glb              # 3D Plant Cell Model
├── single neuron.glb           # 3D Neural Specimen Model
├── white blood cell.glb        # 3D Immune Specimen Model
│
├── animal-cell.png             # UI Selection Icons
├── muscle-cell.png             
├── plant-cell.png              
├── single-neuron.png           
├── white-blood-cell.png        
│
├── root animal cell.png        # Anatomical Location Diagrams
├── root human cell.png
